a sequence of characters
CDATA is intended for human readable data. It is a sequence of characters from the document character set.
Linefeeds are ignored. Carriage returns and tabs are replaced by a single space. When used as an ID or NAME, it must start with a letter and possibly followed by any number of letters, digits, hyphens, underscores, colons, or periods.
